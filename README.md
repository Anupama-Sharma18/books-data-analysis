# 📚 Books Data Analysis using Web Scraping

## 📌 Project Overview
This project extracts book data from a website and analyzes it to understand patterns in ratings and pricing.

## 🛠️ Technologies Used
- Python
- BeautifulSoup
- Pandas
- Matplotlib

## 📊 Features
- Web scraping of book data
- Data cleaning and transformation
- Rating extraction from HTML
- Data visualization

## 📈 Insights
- Most books have lower ratings
- Price is not strongly related to rating
- Prices are mostly in mid-range

## 🚀 How to Run
1. Install dependencies:
   pip install requests beautifulsoup4 pandas matplotlib

2. Run:
   python scraper.py
